Identifiant Amazon Associates

Cet identifiant est ajouté automatiquement à tous les liens Amazon des produits. Si tu changes d'identifiant (nouveau compte, autre région), modifie-le ici une seule fois — il s'applique partout.

Format Amazon : ton-pseudo-XX (XX = 20 pour les USA, 21 pour la France, etc.)

Sauvegarde / restauration complète

Pour transférer toutes tes données vers un autre navigateur, ou faire un backup.

Comment ça marche

Les modifications sont sauvegardées dans le navigateur en temps réel. Pour qu'elles apparaissent sur la page publique, tu dois :

1. Cliquer sur Télécharger amazon-data.json en haut
2. Remplacer le fichier amazon-data.json sur ton site avec le nouveau
3. La page publique chargera automatiquement la nouvelle version
