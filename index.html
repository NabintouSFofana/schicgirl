Tech behind the brand · 2024 — present

The system behind a natural hair brand, built solo.

Schicgirl™ is a bilingual natural hair brand I founded in 2024. This repository holds the technical work that runs it — six small apps, five password-protected admin dashboards, one live AI assistant on the Anthropic Claude API. Everything below is built and maintained by me. The live brand lives at link.schicgirl.me.

How it's built

Stack

  • HTML · CSS · vanilla JavaScript
  • Anthropic Claude API for CoilCare
  • localStorage for per-browser state
  • SHA-256 password hashing via WebCrypto
  • No frameworks · no build step · no backend
  • Hosted via custom domain (not GitHub Pages)

What I built end-to-end

  • Brand: name, trademark, palette, voice
  • Bilingual copy (FR/EN) for every page
  • 5 admin dashboards I use daily
  • LLM prompt design & integration
  • CSV export, JSON backup/restore
  • Customer support, in two languages
