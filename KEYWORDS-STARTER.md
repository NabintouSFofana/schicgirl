# Schicgirl — Starter Keyword List (FR + EN)

**How to read this:** each keyword is a phrase people type into Google. "Difficulty"
is an estimate based on how broad/competitive it is — **🟢 Easy = specific, low
competition, target these FIRST** (you can realistically reach page 1 / #1 in weeks–
months). 🟡 Medium = a few months. 🔴 Hard = broad "head" terms, 6–12+ months, need
backlinks — aspirational, don't start here.

> ⚠️ These are research-informed suggestions, NOT guaranteed search volumes (I don't
> have a paid keyword tool). **Validate each one**: type it into Google and watch the
> autocomplete + "People also ask" — if Google suggests it, real people search it.
> After a few weeks, Search Console → Performance shows your *actual* winning queries.

**How to use a keyword once you pick it:** put it in the page's `<title>`, the main
`<h1>` heading, and naturally a few times in the text. One keyword = one page/post.

### The rule of thumb
Spend 80% of your effort on the 🟢 **Easy** rows for the next 3 months. Win those,
earn backlinks, and the 🟡/🔴 terms gradually become reachable. Chasing 🔴 head terms
on day one is the #1 way beginners burn out with nothing to show.
