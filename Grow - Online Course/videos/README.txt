SCHICGIRL COURSE VIDEOS — how to add them
==========================================

The course (Grow_Course_EN.html) shows a video player on the welcome page and on
every lesson. Each player looks for an MP4 in THIS folder, named by lesson id.
Until the file exists, an elegant "Watch SCHICGIRL teach this lesson" placeholder
shows instead — so the course looks finished even before you record everything.

HOW TO ENABLE A VIDEO
---------------------
1. Record the lesson with SCHICGIRL in HeyGen (scripts are in
   ../SCHICGIRL-Course-Video-Scripts-EN.md or the AI-SPOKESWOMAN folder).
2. Export it as MP4, 16:9 (landscape) is best for a course player.
3. Drop it in this folder with the EXACT filename below.
4. (Optional) Add a matching .jpg with the same name = the thumbnail/poster.

welcome.mp4     -> Welcome to Grow
l1.mp4          -> How Type 4 Hair Actually Grows

INSTRUCTOR PHOTO
----------------
Put SCHICGIRL's portrait in the COURSE folder (one level up) as:
   instructor.jpg
It appears in the "Your instructor: SCHICGIRL" card on the welcome page.
If it's missing, the card shows an "SG" monogram instead.

TIP: you don't need all videos at once. Start with welcome.mp4 + l1.mp4, publish,
and add the rest over time — the course works at every stage.
